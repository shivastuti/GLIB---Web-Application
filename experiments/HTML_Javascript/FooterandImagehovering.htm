Image Animation Effects

Description

In this experiment I tried exploring the animation effects of hovering on an image and to incorporate smooth transitioning of two images when the mouse hovers over it.Also I added the footer to the webpage outside of the Homepage wrapper class.

The given code performs the styling: 



.mainpage_footer {
    clear: both;
    padding: 10px;
    background: #c5faf5;;
    overflow: hidden;
    text-align: center;
    color: black;
    position: absolute;
    top: 140%;
    width: 705px;
    -webkit-box-shadow: 0 0 20px #3cdba0;
    box-shadow: 0 0 10px #3cdba0;
    border: 1px solid black;      
}
.MainPage_Pic1 {
    position: absolute;
    right: 50%;
    margin-top:50px;    
}
    .MainPage_Pic1 img {
        position: absolute;        
        right:100%;
        -webkit-transition: opacity 1s ease-in-out;
        -moz-transition: opacity 1s ease-in-out;
        -o-transition: opacity 1s ease-in-out;
        transition: opacity 1s ease-in-out;
        -webkit-box-shadow: 0 0 20px #3cdba0;
        box-shadow: 0 0 10px #000;
        border: 1px solid gray;
    }
#background_image {
    z-index: -1;
}
    #background_image:hover {
        opacity: 0;
    }
