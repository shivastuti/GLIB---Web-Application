Media Queries,Centered Content and Side Bars

Description

This experiment showcases the use of media queries which resize the contents of the webpage based on the media on which it is being browsed. Also this experiment highlights the centered content along with a sidebar and how it responds to different media sizes. The following code consists of media queries of responsiveness: 
@media (min-width: 50px) and (max-width: 400px) {
    .main {
        background-color : red;
    }
}

@media (min-width: 401px) and (max-width: 800px) {
    .main {
        background-color : green;
        height : 200px;
    }
}

@media (min-width: 801px) {
    .main {
        background-color : yellow;
    }
}
 The following code formats the main content to be centered and responsive along with the sidebars. 

    .wrapper {
            
            width : 800px;
margin: 0 auto;
      
            }
.sidebar {
                float : left;
                width : 200px;
            }
    .main {float:left;
            font-size:20px;
            width : 600px;  

    
}
